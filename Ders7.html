Ders 7: Fonksiyon Yapısı

Belirli bir işlemi yapmak için tasarlanmış kod bloklarına fonksiyon denir.Bir fonksiyonun yapısı

function fonksiyonun adı (parametre1, parametre2,...,parametreN)
{
İŞLEMLER
} şeklindedir.

Parametresiz Fonksiyonlar

Değişken,nesne vb. kullanmadan oluşturulan fonksiyonlardır.

Parametreli Fonksiyonlar



Fonksiyonlarda geri dönüş

return anahtar sözcüğünün iki önemli işlevi vardır:

1.fonksiyonun geri dönüş değerini oluşturur.
2.fonksiyonu sonlandırır

function fonksiyonun adı (argüman1, argüman2,....,argümanN)
{
İŞLEMLER
return değer
}
